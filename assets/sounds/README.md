This directory contains sound effects for the Memorize Numbers game.

The following sound files should be placed here:
1. click.wav - Sound played when user clicks on a number button
2. number_display.wav - Sound played when a number is displayed in the sequence
3. win.wav - Sound played when user passes a level
4. lose.wav - Sound played when user fails a level or game ends
5. level_up.wav - Sound played when advancing to next level
6. error.wav - Sound played when user enters wrong number
7. countdown.wav - Sound played for countdown timer warning

You can download free sound effects from:
- Freesound.org
- Zapsplat.com
- Pixabay.com
- Or use any royalty-free sound effects library

Convert all sounds to WAV format with 44100 Hz sample rate for best compatibility.
